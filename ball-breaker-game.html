Ball Breaker Game

This project is an arcade-style game implemented using an FPGA with Verilog/VHDL.

Project Highlights:

  • Designed using an FPGA (Field Programmable Gate Array)
  • Uses Verilog/VHDL for game logic and display rendering
  • Real-time physics simulation for ball movement

The project demonstrates how gaming logic can be implemented at the hardware level using FPGA technology.
