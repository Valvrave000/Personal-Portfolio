Human Monitoring System for Shipyard Personnel

This project focuses on fall and unconsciousness detection using Raspberry Pi Zero W and sensors.

Key Features:

  • Uses accelerometers and gyroscopes for motion analysis
  • Employs machine learning models for activity classification
  • Real-time monitoring and emergency alerts

The system is designed to enhance safety in high-risk environments such as shipyards.
