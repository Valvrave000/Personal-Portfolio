Leaf Disease Detection

This project utilizes Python, Raspberry Pi Zero, and a Raspberry Pi camera to detect plant diseases using machine learning and image processing techniques.

Key Features:

  • Image preprocessing with OpenCV
  • Feature extraction using deep learning
  • Classification of diseases using a trained neural network

The system provides early detection of plant diseases, helping farmers take preventive measures.
